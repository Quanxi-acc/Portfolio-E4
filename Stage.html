Stage chez OOGARDEN - Ambérieu-en-Bugey

Durée : 10 juin - 25 juillet 2025 (7 semaines)

Mission principale : Créer un programme qui se connecte à Google Analytics 4 pour aller récupérer des données récoltées sur nos sites web (ex : nombre d'utilisateurs par mois et par pays).

Journal de Bord Hebdomadaire

Semaine 1 (10-14 juin 2025)

Semaine 1/7

Objectifs : Prise en main de l'environnement et analyse du code existant

  • - Découverte de l'entreprise et de l'équipe.
  • - Installation et configuration de l'environnement de développement.
  • - Découverte du code Python existant.
  • - Lectures de la documentation Google Analytics API.
  • - Découverte de la version graphique de Git BASH ( Git Extension ).
  • - Découverte du language C#.
  • - Première utilisation de l'outil Visual Studio.
  • - Création d'un depos git en local avec un serveur distant.
  • - Traduction faite de Python à C# ( Code presque déjà présent sur Google merci StackOverflow ).
  • - Identification des pièces manquante pour que mon programme marche (Clé JSON regroupant les identifiants pour récupérer les résultats + numéro de la propriété du compte GoogleAnalytics ).
  • - Premier résultat de mon programme C# dans la console (Nb d'utilisateurs actifs par ville ).
  • - Amélioration des résultats en triant par les 3 meilleurs villes avec l'ajout d'une nouvelle méthode générant le total.
  • - Ajout d'un système de renvoie en document excel.
  • - Nouvelle formation de git avec git Enxtension et comment gérer son dépot & comment l'utiliser en entreprise.
  • - J'ai appris à créer mon propre .gitignore pour améiorer le visuel de mes commit et garder que mes fichiers sources et virer les .dll, .exe, .bin , ...
  • - Rédaction d'une procédure Word pour aider dans le future mes collegues pour refaire la procédure de connexion à Google Analytics ( Création copte de service, propertyID, Coment générer une Clé JSON). PS : Je les ai pas écouté et je l'ai fais en Markdown vous inquiétez pas M. Januel
