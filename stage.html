Stage chez OOGARDEN - Ambérieu-en-Bugey

Durée : 10 Mai - 25 Juin 2025 (7 semaines)

Mission principale : Créer un programme qui se connecte à Google Analytics 4 pour aller récupérer des données récoltées sur les sites web d'OOGarden (ex : nombre d'utilisateurs par mois et par pays).

Technologies : Visual Studio | Python | C# | Google Analytics API

Journal de Bord Hebdomadaire

Semaine 1/7
  • Découverte de l'entreprise et de l'équipe.
  • Installation et configuration de l'environnement de développement.
  • Découverte du code Python existant.
  • Lectures de la documentation Google Analytics API.
  • Découverte de la version graphique de Git BASH (Git Extension).
  • Découverte du langage C#.
  • Première utilisation de l'outil Visual Studio.
  • Création d'un dépot git en local avec un serveur distant.
  • Traduction faite de Python à C# (Code presque déjà présent sur Google merci StackOverflow).
  • Identification des pièces manquantes pour que mon programme marche (Clé JSON regroupant les identifiants pour récupérer les résultats + numéro de la propriété du compte GoogleAnalytics).
  • Premier résultat de mon programme C# dans la console (Nb d'utilisateurs actifs par ville).
  • Amélioration des résultats en triant par les 3 meilleurs villes avec l'ajout d'une nouvelle méthode générant le total.

Semaine 2 ( 18/05/2025 → 24/05/2025 )

Semaine 2/7
  • J'ai appris la méthode de programmation de l'entreprise et renommé toute mes variables en anglais pour qu'elle soit le plus claire possible.
  • Amélioration de mon code en le rendant plus lisible et en essayant de tout remplacer par des fonctions. (pour le moment pas réussi c'est trop dur)
  • Ajout d'une dateRange sur mon fichier Excel qui est définie par les arguments que j'ai donné en entré dans mon main (args[0] -> startDate args[1] -> endDate) (période de collecte des résultats)
  • Mise à jour de ma Procédure d'utilisation
  • J'ai mis de nouvelles valeurs dans la collecte des données de mon code pour qu'il puisse collecter des données qui sont vraiment importantes et utiles pour l'entreprise (activeUsers, newUsers, bounceRate, screenPageViews, etc..)
  • Récupération des données sur les différents sites de OOGarden hors France (Belgique et Allemagne)
  • Ajout d'un système de tri pour avoir une cohérence dans les résultats obtenu dans mon fichier excel

Semaine 3 ( 25/05/2025 → 31/05/2025 )

Semaine 3/7
  • Découverte d'un Web Service
  • Découverte du fonctionnement d'un Batch
  • Apprentissage de la création d'un appel à Sage ( Logiciel qui va recevoir mes données de GA4 ) en XML Soap /li>

Mon projet évoluant et devenant plus complexe et compliqué, il y a un ajout d'un accompagnement où chaque jour pendant 30 minutes mes tuteurs viennent avec moi et m'orientent sur les travaux que j'ai à faire, ce qui me reste à faire avec correction de mon travail, etc...
  • Correction de mon taux de rebond qui renvoyait un mauvais total, il faisait la somme de l'ensemble des pourcentages de chaques pays sous forme décimal ce qui est mathématiquement faux.
