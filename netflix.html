Détails du projet Netflix

Reproduction d'un site de streaming

Contexte : Projet de développement web visant à recréer les fonctionnalités principales d'une plateforme de streaming.

Technologies : JavaScript, HTML5, CSS, PHP, MySQL

Espace Utilisateur

Authentification

  • Création de compte via la page "Mon Compte"
  • Connexion via une page dédiée
  • Vérification d'email unique
  • Adaptation pour téléphone

Navigation & Contenu

  • Carrousels interactifs pour films/séries
  • Filtrage par genre et type
  • Barre de recherche dynamique
  • Bande-annonce pour chaque film

Espace Administrateur

Gestion du Catalogue

  • Ajout/suppression de films et séries
  • Gestion des genres et catégories
  • Intégration directe dans la base de données
